# GTT Bot — Example Queries

A reference guide for getting the most out of GTT Bot's knowledge base search.

---

## Tips for better results

The search is semantic — it understands meaning, not just keywords. Use **specific concepts and terminology** rather than questions.

| Less effective | More effective |
|---|---|
| `what is DIF` | `deterministic intent folding merly architecture` |
| `how does the bot work` | `RAG vector search obsidian knowledge base` |
| `tell me about creativity` | `creativity sociocultural context collective problem solving` |

## Commands reference

| Command | Privacy | Visible to mods | Use when |
|---|---|---|---|
| `/knowledge-search <query>` | Private thread | Yes | You want a searchable, deletable record |
| `/knowledge-base <query>` | DM only | No | You want fully private results |
| `@GTT Bot <question>` | Public thread | Yes | You want a GTT-voiced answer with reasoning |
| `/glossary [term]` | Ephemeral | No | Learn GTT terminology and find example queries |
| `/status` | Ephemeral | No | Check bot health and config |
| `/thread-mode on/off` | Ephemeral | No | Toggle thread replies for `@GTT Bot` |
